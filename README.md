\# Apple Grading System MVP



\## Objective

Develop an AI-based apple grading system for agricultural and food quality analysis.



The system performs:



1\. Fresh vs Rotten classification

2\. Apple quality analysis

3\. Grade assignment (A/B/C)



\## Workflow



Apple Image

↓

CNN Freshness Detection

↓

Quality Analysis

(Size, Weight, Sweetness, Juiciness)

↓

Final Grade Assignment



\## Datasets Used



1\. Fruit Freshness Dataset

\- Fresh apples

\- Rotten apples



2\. Apple Quality Dataset

Features:

\- Size

\- Weight

\- Sweetness

\- Crunchiness

\- Juiciness

\- Ripeness

\- Acidity
